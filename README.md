# Meal Planner Desktop Application

This is a Meal Planner desktop application developed using the Electron framework with HTML, CSS, and JavaScript.

## Features
- Fetches meal data from TheMealDB API
- Displays meal details including ingredients, instructions, and images
- Identifies vegetarian meals
- Estimates meal calories
- Implements full CRUD functionality for meal planning
- Multi-page desktop application

## API Used
- https://www.themealdb.com/api.php

## How to Run
1. Install Node.js
2. Open terminal in the project folder
3. Run:
   npm install
   npm start
